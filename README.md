# 🍔 Burger JS<br>

Progetto interattivo per la gestione di un menu **Burger** sviluppato con **HTML, CSS e JavaScript**.  <br><br>
Utilizza array e funzioni avanzate per manipolare i dati:<br>

- `forEach` → per iterare sugli elementi del menu  <br>
- `map` → per trasformare gli elementi  <br>
- `filter` → per filtrare prodotti <br> 
- `reduce` → per calcolare totali  <br><br>

## Tecnologie<br>
- HTML<br>
- CSS<br> 
- JavaScript
